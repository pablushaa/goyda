# 🗣️🔥 GOYDA
Tool that generates pro-russian statements 

## 📩 Installing

From sources:
```sh
git clone https://github.com/zvosolider/goyda
cd goyda
make install
```

## 🗑️ Uninstalling
```sh
make uninstall
```

## ⚙️ Using
Generating 10 words:
```sh
goyda 10
```
Output:
```sh
ПУТИН ЦАРЬ ГОЙДА ZOV ZZZVVVOOO zov zov O Я РУССКИЙ ПУТИН ЦАРЬ ПУТИН ЦАРЬ
```
